source code  
Initialization of stacking project with specified properties:
    * **project** - unique name of stacking  project
    * **split_ratio** - ratio for split data by folds
    * **path_to_data** - Path to initial data
    * **path_to_test** - Path to test data
    * **store_format** - Format of how data is stored (One of the following:  'json','excel','csv')
:return: Function is setting global variables; no returns
